# MolecuLab

An interactive chemistry learning sandbox for students.

## Overview

MolecuLab is a comprehensive web-based chemistry exploration tool that provides an engaging, interactive environment for learning fundamental chemistry concepts through hands-on experimentation.

## Features

### 9 Interactive Modules:
1. **Periodic Table Explorer** - Interactive periodic table with element details and trend visualizations
2. **Bonding & Molecules** - Drag-and-drop molecule builder with bond validation
3. **Chemical Equations & Reactions** - Equation balancer and reaction type classifier
4. **Stoichiometry Calculator** - Step-by-step stoichiometry problem solver
5. **Acids & Bases** - pH calculator, titration curves, and buffer systems
6. **Gas Laws Simulator** - Interactive gas law visualizations and calculations
7. **Thermochemistry** - Energy diagrams, specific heat calculator, and calorimetry
8. **Organic Chemistry** - Functional groups, biological macromolecules, and IUPAC naming
9. **Study Tools** - Flashcards and memorization aids

## Tech Stack

- React (Vite)
- Tailwind CSS
- D3.js (data visualizations)
- Chart.js (graphs and charts)
- Framer Motion (animations)
- Zustand (state management)

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/moleculab.git

# Navigate to project directory
cd moleculab

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

## Deployment

This project is configured for deployment on Cloudflare Pages.

## Educational Context

MolecuLab is designed for students with limited chemistry background, providing a safe, non-judgmental space to explore chemistry concepts. The sandbox approach allows unlimited experimentation without scoring or progress tracking, reducing anxiety and building confidence.
